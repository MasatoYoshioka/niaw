Application クラス
-ルートディレクトリの指定
-アクションに合わせたルーティング定義
-接続するデータベースの指定(DbManagerクラスを操作)

-データベースアクセス処理の実装

-アクションにあわせたHTMLの記述
-レイアウトファイルの記述
